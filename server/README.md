# Tsunderizer Backend

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file in this directory with your Anthropic API key:
   ```env
   ANTHROPIC_API_KEY=your-anthropic-api-key-here
   ```

3. Start the server:
   ```bash
   npm start
   ```

The server will run on port 5000 by default.
